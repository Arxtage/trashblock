TrashBlock - Privacy Policy

Last updated: March 5, 2026

Overview

TrashBlock is a Chrome extension that blocks distracting websites by requiring a typing challenge to unlock them. It is designed to be completely private and offline.

Data Collection

TrashBlock does not collect, transmit, or share any data. Specifically:

  • No personal information is collected
  • No browsing history is tracked or recorded
  • No analytics or telemetry of any kind
  • No cookies are set
  • No network requests are made to external servers
  • No third-party services or SDKs are used

Data Storage

All data (your blocked sites list, unlock phrase, active days, and temporary unlock status) is stored locally on your device using Chrome's chrome.storage.local API. This data never leaves your browser.

Permissions

TrashBlock requests the following permissions:

  • declarativeNetRequest - Used to redirect blocked sites to the challenge page
  • storage - Used to save your settings locally
  • alarms - Used to re-block sites after the temporary unlock expires
  • webNavigation - Used as a fallback to detect navigation to blocked sites
  • Host permissions (<all_urls>) - Required so the extension can block any site you add to your list. No data from these sites is collected or transmitted.

Changes to This Policy

If this policy is updated, the changes will be posted on this page with an updated date.
